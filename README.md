# FinBot — Natural Language Interface for Finance Infrastructure

A Python-based Finance Chatbot using keyword-based NLP.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Start the web server:
   python app.py

3. Open in browser:
   http://localhost:5000

## CLI Mode

   python chatbot_engine.py

## Tech Stack
- Python 3 + Flask
- HTML, CSS, JavaScript
- Keyword-based NLP (no ML model)
